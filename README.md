FuturePath is an AI-powered career guidance and project-based learning platform that helps students discover suitable careers, real-world problems, and project ideas based on their interests and access.

This repository focuses on **Module 6: AI Analysis & Results**, the core intelligence of the system.

## What This Module Does
- Analyzes completed student assessments  
- Uses AI to generate personalized career insights  
- Recommends career paths, industries, and projects  
- Displays results in a clear and actionable UI  

## Main Features
- AI-powered career analysis  
- RIASEC interest-based results  
- Career and industry recommendations  
- Project ideas and local opportunities  
- Responsive modern UI  

## Assessments Used
- RIASEC Career Assessment  
- Unique Access Assessment  
- Problems Assessment  

## Tech Stack
- Frontend: React  
- Styling: Tailwind CSS  
- UI Components: shadcn/ui  
- AI: Large Language Model (LLM)
